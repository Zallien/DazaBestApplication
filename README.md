# 🚀 Computerized Point of Sale with Inventory Management System

> A desktop-based POS and inventory system designed for Daza’s Best to streamline transactions and manage stock efficiently.

---

## 📌 Overview

This project is a **Point of Sale (POS) and Inventory Management System** built using **C# with Windows Forms**.  
It helps the Daza’s Best team **digitize daily transactions, improve accuracy, and speed up operations**.

---

## 🛠️ Tech Stack

- **Frontend:** .NET Framework, Bunifu UI  
- **Backend:** C#  
- **Database:** SQLite  
- **Tools:** Git, Visual Studio, SQLite Viewer  

---

## ✨ Features

- ✅ **User Authentication** (Secure login system)  
- ✅ **Admin Dashboard** (Overview of sales and system data)  
- ✅ **Account Management** (Manage users and roles)  
- ✅ **Inventory Management** (Track and update stock)  
- ✅ **Point of Sale (POS)** (Fast and efficient transactions)  
- ✅ **Backup System** (Multiple data backup options)
